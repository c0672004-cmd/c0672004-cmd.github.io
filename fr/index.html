Projects des données

Analyse de la Performance : Vrindavan Retail

Analyse de Données de Bout en Bout | Excel Avancé & Tableaux de Bord

🛍️ Le Défi

Le magasin gérait plus de 31 000 transactions désordonnées. Les tendances de ventes étaient floues et les données démographiques manquaient, rendant la planification des stocks difficile.

💡 La Solution

J'ai conçu un tableau de bord interactif pour suivre le Panier Moyen (690 $), identifier mars comme le mois record, et découvrir que 64 % du chiffre d'affaires provient de la clientèle féminine.

Approche Technique

  • ✔ Intégrité des Données : Utilisation des fonctions SWITCH et VALEUR pour transformer les quantités textuelles en entiers numériques pour des calculs précis.
  • ✔ Logique Temporelle : Création d'un index de tri chronologique pour assurer une analyse cohérente des tendances de janvier à décembre.
  • ✔ Stratégie de Visualisation : Mise en œuvre d'un graphique combiné à deux axes pour corréler le Chiffre d'Affaires (Millions) et le Volume de Commandes (Milliers).

🏅 Succès Olympique : Étude Statistique

Projet de recherche multivariée analysant l'influence de la richesse nationale (PIB) et de la démographie sur la performance athlétique. Cette étude utilise des statistiques descriptives et des modèles de régression pour identifier les moteurs de la réussite sportive internationale.

  • Résultats Clés : Le PIB national montre une forte corrélation positive avec le nombre de médailles (r = 0,769). À l'inverse, la taille de la population présente une relation négative dans les modèles prédictifs, soulignant que l'allocation des ressources prime sur le volume démographique.
  • Indice d'Efficacité : Création d'un indice personnalisé pour mettre en lumière les nations (ex: Dominique, Jamaïque) qui surperforment par rapport à leur base économique.
  • Rigueur Statistique : Utilisation de tests ANOVA pour les comparaisons régionales et de tests VIF pour garantir la fiabilité des coefficients de régression.

Outils : R, Tidyverse, ANOVA, Analyse VIF, ggplot2, Tests d'Hypothèses

🍷 Prédiction de la Qualité du Vin : Analyse Python

Développement d'un optimiseur de descente de gradient "from scratch" en Python. Ce projet analyse les relations non-linéaires entre les variables physico-chimiques à travers une étude comparative de modèles de régression linéaire et polynomiale.

  • Résultats Clés : L'étude démontre la nature non-linéaire des données ; le modèle polynomial de degré 2 (R2 = 0,428) surpasse nettement l'approche linéaire initiale.

Outils : Python, NumPy, Pandas, Matplotlib, Descente de Gradient, Modélisation Statistique

🤖 Classification de Chiffres MNIST : Machine Learning & Deep Learning

Étude comparative de cinq algorithmes — Régression Logistique, LDA, Arbres de Décision, Random Forest et Réseaux de Neurones — pour classer les chiffres manuscrits de la base de données MNIST. Le projet explore l'équilibre entre efficacité computationnelle et précision prédictive.

  • Optimisation : Obtention d'une **précision de 99,17 %** sur l'ensemble de test en optimisant un CNN (architecture LeNet-5 modifiée) avec Batch Normalization, activations ReLU et couches de Dropout.
  • Techniques Avancées : Mise en œuvre de l'augmentation de données (rotation, zoom, décalage) et d'un planificateur de taux d'apprentissage variable pour améliorer la robustesse du modèle.
  • Analyse Comparative : Réalisation d'une validation croisée (5-fold) sur tous les modèles de base pour évaluer les compromis entre simplicité linéaire et complexité non-linéaire.

Outils : Python, PyTorch, Scikit-learn, CNN (LeNet-5), Augmentation de Données, Optimisation des Hyperparamètres
