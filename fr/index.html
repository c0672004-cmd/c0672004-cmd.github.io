Vrindavan Retail Performance Analysis

Analyse de la performance d'un commerce de détail à travers le nettoyage de données et la visualisation de tableaux de bord.

End-to-End Data Analysis | Advanced Excel & Dashboarding

🛍️ The Challenge

The store operated with over 31,000 messy transactions. Sales trends were unclear, and demographic insights were missing, making inventory planning and financial forecasting difficult.

💡 The Solution

I built an interactive dashboard to track Average Order Value ($690), identify March as the peak month, and discover that 64% of revenue comes from female customers.

Technical Deep Dive

  • ✔ Data Integrity: Used SWITCH and VALUE logic to convert string-based quantities (e.g., "One") into numeric integers for accurate summation and statistical modeling.
  • ✔ Temporal Logic: Engineered a chronological sorting index (Jan-Dec) using custom date-mapping to allow for meaningful time-series trend analysis.
  • ✔ Visual Communication: Implemented a dual-axis combo chart to visualize the correlation between Revenue (Millions) and Order Volume (Thousands) on a single unified scale.

🤖 Classification de Chiffres MNIST : Machine Learning & Deep Learning

Étude comparative de cinq algorithmes — Régression Logistique, LDA, Arbres de Décision, Random Forest et Réseaux de Neurones — pour classer les chiffres manuscrits de la base de données MNIST. Le projet explore l'équilibre entre efficacité computationnelle et précision prédictive.

  • Optimisation : Obtention d'une **précision de 99,17 %** sur l'ensemble de test en optimisant un CNN (architecture LeNet-5 modifiée) avec Batch Normalization, activations ReLU et couches de Dropout.
  • Techniques Avancées : Mise en œuvre de l'augmentation de données (rotation, zoom, décalage) et d'un planificateur de taux d'apprentissage variable pour améliorer la robustesse du modèle.
  • Analyse Comparative : Réalisation d'une validation croisée (5-fold) sur tous les modèles de base pour évaluer les compromis entre simplicité linéaire et complexité non-linéaire.

Outils : Python, PyTorch, Scikit-learn, CNN (LeNet-5), Augmentation de Données, Optimisation des Hyperparamètres
